XML
 21. Создать внешний репозиторий c названием XML.        		  	open https://github.com/ and login,
						            			click  create  "New" Repositories, Repository name "XML" and public
 22. Клонировать репозиторий XML на локальный компьютер.    			git clone https://github.com/RuslanRS13/XML.git
 23. Внутри локального XML создать файл “new.xml”.	    			touch new.xml
 24. Добавить файл под гит.				   			git add new.xml
 25. Закоммитить файл.					    			git commit -m "add file new.xml"
 26. Отправить файл на внешний GitHub репозиторий.	   			git push
 27. Отредактировать содержание файла “new.xml” - написать информацию о себе    
    (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). 	
    Всё написать в формате XML.							vim new.xml
										insert 
										<?xml version="1.0"encoding="UTF-8"?>
										<person>
										<name>Ruslan</name>
										<surname>Samsonov</surname>
										<age>23</age>
										<animals null="true"/>
										<desired salary usd>400</desire_salary_usd>
										</person>

										[ESC] :wq

 28. Отправить изменения на внешний репозиторий.				git commit -m "add info about person"       git push
 29. Создать файл preferences.xml     
 30. В файл preferences.xml добавить информацию о своих предпочтениях 		touch preferences.xmlvim
     (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате XML.
										vim Prefecences.json
										insert
										<?xml version="1.0"encoding="UTF-8"?>
										<interests>
										<favorite_movie>The Lord Of The Rings</favorite_movie>
										<favorite_serial>Chicago PD</favorite_serial>
										<favorite_meal>Scrambled eggs</favorite_meal>
										<favorite_time_of_year>Autumn</favorite_time_of_year>
										<country_you_would_like_to_visit>Canada</country_you_would_like_to_visit>
										</interests>
										[ESC] :wq

 31. Создать файл sklls.xml добавить информацию о скиллах 			touch skills.xml
     которые будут изучены на курсе в формате XML				vim Skills.json
										insert
										<?xml version="1.0'encoding="UTF-8"?>
										<skills>

										[ESC] :wq		

 32. Сделать коммит в одну строку.						
										git commit -m "add personal info preferences,skills.xml"
										
 33. Отправить сразу 2 файла на внешний репозиторий.				git push				
 34. На веб интерфейсе создать файл bug_report.xml.				open https://github.com/RuslanRS13/XML/tree/main 
										click on "add file" then  "add create new file" and "name the file"
 35. Сделать Commit changes (сохранить) изменения на веб интерфейсе.            Commit new file "add bug_report.xml" 

 36. На веб интерфейсе модифицировать файл bug_report.xml, 			open https://github.com/RuslanRS13/XML/blob/main/bug_report.xml
     добавить баг репорт в формате XML.						click on "edit this file"
										<?xml version="1.0"encoding="UTF-8"?>
										<Bug_report>
										<Bug_ID>1</Bug_ID>
										<Title>Товар не добавляется в корзину<Title>
										<STR>
										<step_1>Зайти на страницу товара</step_1>
										<step_2>Нажать кнопку Добавить в корзину</step_2>
										<step_3>Перейти в корзину</step_3>
										</STR>
										<Expected_result>Товар добавлен в корзину</Expected_result>
										<Actual_Result>Корзина Пуста</Actual_Result>
 										</Bug_report>
										
 37. Сделать Commit changes (сохранить) изменения на веб интерфейсе.		commit file "add bug report"
 38. Синхронизировать внешний и локальный репозиторий XML			git pull
